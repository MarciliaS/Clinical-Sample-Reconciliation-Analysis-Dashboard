📊 Clinical Sample Reconciliation & Analysis Dashboard
📌 Overview

This project simulates a clinical sample tracking and reconciliation workflow using Excel. It compares an expected sample roster against received sample data, then uses an analysis layer to identify missing samples, flag records for review, and summarize operational metrics.

🧠 Key Features
Separate source sheets for expected and received sample data
Reconciliation logic using XLOOKUP
Status derivation (Received vs Missing)
Review flag generation for missing records
Summary metrics using COUNTIFS and SUMIFS
Breakdown reporting by study and visit
Pivot Table analysis for operational reporting
📂 Workbook Structure
Expected_Samples → master sample roster
Received_Samples → received sample records
Analysis → reconciliation and reporting layer
Pivot_Analysis → summarized reporting views
📊 Metrics Included
Total expected samples
Total received samples
Missing sample count
Review-required records
Expected vs received quantity totals
Summary by study
Summary by visit
